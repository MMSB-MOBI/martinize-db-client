API

The MAD API enables programmatic access to MAD in unique and advanced ways. Our website produces an API to download molecule representation.
This is the format of the route :
https://mad.ibcp.fr/api/molecule/get/[forcefield]/[molecule_name][.extention]
Parameter Type Default Description
Forcefield string None ... (exemple : martini22, martini3001)
Molecule name string None ... (example : POPE)
File type string .zip You can ask for a specific type of file ('.itp', '.gro', '.pdb') or you can download everything in zip format in this case adds'' or '.zip'. zip)
