Polymer generator

This new MAD feature allows you to generate new molecules (example: polysacharide, Battery polymer) or to modify existing molecules (example: Protein PEGylation, Glycosylation). This feature is powered by Polyply, this program is a python suite designed to facilitate the generation of file and coordinate for molecular dynamic simulation. You can generate a molecule using some of the molecular compounds provided by Polyply by using the Martini3 force field, or you can load your own molecule (from the MAD database, MAD history or upload file directly).

The following example will guide you through the multiple use of MAD:Polymer generator.

1. Create a simple peptide
2. Pegylate a protein
3. Load a molecule from the history and modify it
4. Limitation
